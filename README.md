## Social Mission

toMy Kawiarnia is a project of Fundacja Leny Grochowskiej, creating employment opportunities for people with disabilities. The café combines specialty coffee, artisan pastries, and ceramic workshops.

## Repository Notes

- This repository contains the compiled static site (HTML/CSS/JS) for deployment.
- Assets are located under `assets/`.
- `.nojekyll` ensures GitHub Pages serves files without Jekyll processing.

For content updates, replace images or adjust text in `index.html` and rebundle from source (if available). This repo intentionally omits build tooling details and AI/tool references.
